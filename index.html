URL Shortener Microservice API

This is a simple API that returns a JSON with a shortened url that can be used to access the provided url.

You must pass a valid url (include http or https and at least one period)

Example creation usage:

https://little-url.herokuapp.com/new/https://www.google.com

https://little-url.herokuapp.com/new/http://foo.com:80

Example output:

{ "original_url":"http://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }
