URL Shortener Microservice API

This is a simple API that returns a JSON with a shortened url that can be used to access the provided url.

You must pass a valid url (include http or https and at least one period), use the short url in the address bar to visit the site faster!

Example creation usage:

https://url-shortener-kf.herokuapp.com/new/https://www.google.com

https://url-shortener-kf.herokuapp.com/new/http://foo.com:80

Example output:

{ "original_url":"http://foo.com/this/is/the/site/you/want/to/visit", "short_url":"url-shortener-kf.herokuapp.com/8" }
